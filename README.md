# MenuzinhoMenuzao
App Android Didático: CRUD utilizando menu de opções na toolbar para acesso ao cadastro e menu de contexto flutuante para alteração e exclusão.

O cadastro é acessado pelo menu na barra de opções, na canto superior esquerdo, conforme *Figura 1*.
As funcionalidades de alteração e exclusão são acessadas pelo menu de contexto flutuante que surge ao realizar um clique longo em um item da lista, conforme *Figura 2*.
